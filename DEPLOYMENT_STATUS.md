## 배포 URL 확인 방법

1. **Vercel 대시보드 접속**
   - https://vercel.com/dashboard
   - GitHub 계정으로 로그인

2. **프로젝트 확인**
   - "facd-game" 프로젝트 찾기
   - 프로젝트 클릭

3. **실제 URL 확인**
   - Domains 섹션에서 할당된 URL 확인
   - 일반적인 패턴:
     - `[프로젝트명]-[사용자명].vercel.app`
     - `[프로젝트명]-git-main-[사용자명].vercel.app`
     - 또는 Vercel이 자동 생성한 랜덤 URL
